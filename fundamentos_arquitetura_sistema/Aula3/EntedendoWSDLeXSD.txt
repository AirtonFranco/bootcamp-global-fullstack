ENTEDENDO O QUE É WSLD E XSD

WSLD

- WSLD (Web services Description Language.)
- Usado para descrever Web Services, Funciona como um contrato do serviço
- A descricao é feito em um documento XML, onde é descrito o serviço, especificaçoes de acesso, operaçoes e metódos.

Obs: Contrato do serviço, padronizaçao

XSD

- XSD - XML Schemo Definition
- É um schema mo formato  XML usado para definir a estrutura de dados que será validada no XML
- O XSD Funciona como uma documentacao de como deve ser montado o SOAP Message (XML) que será enviado através de Web Service.

Obs: Regras do serviço
